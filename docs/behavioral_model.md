# Behavioral Model

The research model treats observed behavior as the result of multiple interacting components rather than a single personality label.

Publicly, the key concepts are:
- stable profile structure
- dynamic state under pressure
- short and persistent memory effects
- partial recovery rather than full reset
- differentiated conversational expression under controlled constraints

In practice, this allows the same external scenario to generate distinct trajectories across synthetic agents while preserving auditability and deterministic control.
